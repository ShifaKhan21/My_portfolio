# Shifa Khan - Portfolio Website

This is a modern, 3D animated portfolio website built with React, Three.js, and Vite.

## Features
- **3D Hero Animation**: Floating, rotating cubes with custom tech logos (React, Node, AIML, DSA, CNN).
- **Premium Design**: Dark theme, glassmorphism, and smooth scrolling.
- **Responsive Layout**: Works on desktop and mobile.
- **Interactive Elements**: Start exploring immediately.

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```

## Customization
- Modify `src/App.jsx` to change logos or add more content.
- Update `src/index.css` for styling tweaks.
